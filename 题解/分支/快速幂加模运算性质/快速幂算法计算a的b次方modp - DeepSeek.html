function pow_mod(a, b, p):
    res = 1
    a = a % p
    while b > 0:
        if b % 2 == 1:   # 或 b & 1
            res = (res * a) % p
        a = (a * a) % p
        b = b // 2       # 或 b >>= 1
    return res

好的，我们来系统地总结模运算的基本性质。这些性质在数论、密码学和算法设计中非常重要。

1. 定义

对于整数 a,b,pa, b, pa,b,p（p>0p > 0p>0），

amod  pa \mod pamodp

表示 aaa 除以 ppp 的余数，取值范围是 0,1,…,p−10, 1, \dots, p-10,1,…,p−1。

等价地，amod  p=ra \mod p = ramodp=r 表示：

a=p⋅q+r,0≤r<pa = p \cdot q + r, \quad 0 \le r < pa=p⋅q+r,0≤r<p

其中 qqq 是整数（商）。

同余记号：

a≡b(modp)当且仅当p∣(a−b)a \equiv b \pmod{p} \quad \text{当且仅当} \quad p \mid (a - b)a≡b(modp)当且仅当p∣(a−b)

即 aaa 和 bbb 除以 ppp 的余数相同。

2. 基本算术性质

(1) 加法模运算

(a+b)mod  p=[(amod  p)+(bmod  p)]mod  p(a + b) \mod p = [(a \mod p) + (b \mod p)] \mod p(a+b)modp=[(amodp)+(bmodp)]modp

原因：设 a=pq1+r1a = p q_1 + r_1a=pq1​+r1​，b=pq2+r2b = p q_2 + r_2b=pq2​+r2​，则
a+b=p(q1+q2)+(r1+r2)a + b = p(q_1 + q_2) + (r_1 + r_2)a+b=p(q1​+q2​)+(r1​+r2​)，余数部分是 r1+r2r_1 + r_2r1​+r2​，可能大于等于 ppp，所以最后再取一次模。

(2) 减法模运算

(a−b)mod  p=[(amod  p)−(bmod  p)]mod  p(a - b) \mod p = [(a \mod p) - (b \mod p)] \mod p(a−b)modp=[(amodp)−(bmodp)]modp

若差为负，加 ppp 直到非负且小于 ppp。

等价写法：

(a−b)mod  p=[(amod  p)−(bmod  p)+p]mod  p(a - b) \mod p = [(a \mod p) - (b \mod p) + p] \mod p(a−b)modp=[(amodp)−(bmodp)+p]modp

这样保证括号内非负。

(3) 乘法模运算

(a×b)mod  p=[(amod  p)×(bmod  p)]mod  p(a \times b) \mod p = [(a \mod p) \times (b \mod p)] \mod p(a×b)modp=[(amodp)×(bmodp)]modp

这是快速幂算法的基础。

(4) 乘方模运算

abmod  p=[(amod  p)b]mod  pa^b \mod p = [(a \mod p)^b] \mod pabmodp=[(amodp)b]modp

3. 除法模运算（模逆元）

普通算术中除法：c=a/bc = a / bc=a/b 等价于 c×b=ac \times b = ac×b=a。
在模运算中，模逆元定义：

如果 gcd⁡(b,p)=1\gcd(b, p) = 1gcd(b,p)=1，则存在整数 b−1b^{-1}b−1（模 ppp 意义下）使得：

b⋅b−1≡1(modp)b \cdot b^{-1} \equiv 1 \pmod{p}b⋅b−1≡1(modp)

那么：

(a/b)mod  p=(a×b−1)mod  p(a / b) \mod p = (a \times b^{-1}) \mod p(a/b)modp=(a×b−1)modp

注意：若 gcd⁡(b,p)≠1\gcd(b, p) \neq 1gcd(b,p)=1，则 bbb 在模 ppp 下没有逆元，不能直接做除法。

4. 模逆元的求法

  1. 扩展欧几里得算法：解 bx+py=1b x + p y = 1bx+py=1，则 xxx 是 b−1mod  pb^{-1} \mod pb−1modp。

5. 指数模运算的性质

(1) 模幂的周期性质

由欧拉定理：若 gcd⁡(a,m)=1\gcd(a, m) = 1gcd(a,m)=1，则

aϕ(m)≡1(modm)a^{\phi(m)} \equiv 1 \pmod{m}aϕ(m)≡1(modm)

其中 ϕ(m)\phi(m)ϕ(m) 是欧拉函数（小于 mmm 且与 mmm 互质的正整数个数）。

特例（费马小定理）：若 ppp 为质数且 p∤ap \nmid ap∤a，则

ap−1≡1(modp)a^{p-1} \equiv 1 \pmod{p}ap−1≡1(modp)

因此计算 abmod  ma^b \mod mabmodm 时，可先对指数 bbb 模 ϕ(m)\phi(m)ϕ(m)（当 gcd⁡(a,m)=1\gcd(a, m)=1gcd(a,m)=1）：

abmod  m=abmod  ϕ(m)mod  ma^b \mod m = a^{b \mod \phi(m)} \mod mabmodm=abmodϕ(m)modm

（注意：必须满足 gcd⁡(a,m)=1\gcd(a, m)=1gcd(a,m)=1 时才可直接对指数取模；不互质时需小心，可能要用其他方法。）

(2) 模运算没有直接的“指数取模分配律”

错误：abmod  p≠(amod  p)(bmod  p)mod  pa^{b} \mod p \neq (a \mod p)^{(b \mod p)} \mod pabmodp=(amodp)(bmodp)modp。
正确：底数可随时取模，指数一般不能直接取模（除非用欧拉定理化简指数，且底数与模数互质时）。

6. 其他重要性质

  • 结合律、交换律：模 ppp 下的加法和乘法满足结合律、交换律，有单位元（0 和 1）。

  • 分配律：乘法对加法满足分配律。

  • 消去律：如果 gcd⁡(c,p)=1\gcd(c, p) = 1gcd(c,p)=1，那么
